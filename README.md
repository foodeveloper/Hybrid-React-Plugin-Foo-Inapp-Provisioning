# fooapplewalletplugin

FooAppleWallet react plugin

## Installation

```sh
pod 'FooAppleWalletPlugin'
pod 'FooAppleWallet/Debug', :modular_headers => true
```

## Usage

```js
import { multiply } from 'fooapplewalletplugin';

// ...

const result = await multiply(3, 7);
```
